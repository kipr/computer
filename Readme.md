computer
========

Computer is a TCP-based target for KISS IDE.

Requirements
============

* Qt 4.7 or higher
* CMake 2.8.0 or higher
* easydevice
* pcompiler
* libkar

Building
========

	mkdir build
	cd build
	cmake ..
	make
